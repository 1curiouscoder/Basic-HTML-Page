What is HTML?

HTML stands for Hypertext Markup Language which specifies various elements like <p>, <h1>, etc. to define the type of content so the browser can display it accordingly.

HTML Elements

  • HTML elements are defined by a starting tag , some content and then a closing tag.

  • They are used to specify different type of content and help browsers define how to display a content.

  • Basic syntax of a HTML element - <tagname> Content goes here... </tagname>
    Following is a syntax for a paragraph element:

    <p> Content comes here </p>

  • The element that don't have any content are called empty elements. eg. <br>, <hr>, etc.

HTML Attributes

  • HTML element attribute provide additional information about the element.

  • Attribute usually come in name/value pairs: name="value"

    • Eg. id="head"

Href Attribute

Href attribute sprecies the url to which a link points or will head to once clicked. Eg. href="https://www.w3schools.com"

Src Attribute

src attribute sprecies the location of the file to be displayed. Eg. src="https://www.w3schools.com/images/image.jpg"
